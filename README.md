# sarra-cli

Daily developer cli tools in one place

## Overview

sarra-cli is a comprehensive command-line interface that brings together essential developer tools and workflows into a single, unified platform.

## Commands

Below is a list of all available commands. Click on each command to view detailed documentation:

### Core Commands

- [**init**](src/commands/init/help.md) - Initialize a new project with sarra-cli configuration
- [**config**](src/commands/config/help.md) - Manage sarra-cli configuration settings
- [**run**](src/commands/run/help.md) - Execute development tasks and scripts

## Installation

```bash
# Installation instructions coming soon
npm install -g sarra-cli
```

## Quick Start

```bash
# Initialize a new project
sarra init

# Configure your settings
sarra config set editor vim

# Run a development task
sarra run dev
```
